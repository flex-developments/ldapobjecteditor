# ldapobjecteditor
Java - Script para modificación en lote de atributos en objetos de un LDAP o ActiveDirectory. Específicamente para activación o desactivación de usuarios.

## Dependencias
 * lib-flex-helpers v010
   * commons-net-3.1
   * mail
